# Proyecto POO Java

## Descripción
Este proyecto consiste en un manual web académico sobre Programación Orientada a Objetos (POO) en Java.

El sitio contiene información teórica y práctica sobre diferentes temas fundamentales de POO, incluyendo ejemplos resueltos y ejercicios propuestos.

---

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Git
- GitHub
- GitHub Pages
- Java

---

## Temas desarrollados

1. Clases Abstractas e Interfaces
2. Composición y Modularización
3. Manejo de Errores y Excepciones
4. Colecciones y ArrayList
5. Archivos y Persistencia
6. Principios SOLID
7. Introducción a POO y Entorno Java
8. Clases y Objetos
9. Encapsulamiento
10. Constructores y toString()
11. Herencia en Java
12. Polimorfismo

---

## Características

- Landing page funcional
- Diseño responsive
- Navegación por secciones
- Ejemplos en Java
- Ejercicios prácticos
