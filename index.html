Programación Orientada a Objetos

Sitio académico sobre los temas fundamentales de POO en Java.

1. Clases y Objetos

Las clases son plantillas que permiten crear objetos. Los objetos representan entidades con atributos y métodos.

Ejemplo 1


class Persona {

    String nombre;

    void saludar() {
        System.out.println("Hola");
    }
}

Ejemplo 2


class Vehiculo {

    String marca;
    int velocidad;
}

Ejercicios

  • Crear una clase Estudiante con atributos y métodos.
  • Diseñar una clase Producto para una tienda virtual.

2. Composición y Modularización

La composición representa relaciones entre objetos. Permite construir sistemas más organizados y reutilizables.

Ejemplo 1


class Computador {

    Procesador procesador;
}

Ejemplo 2


class Casa {

    Habitacion habitacion;
}

Ejercicios

  • Crear una clase Auto con una clase Motor.
  • Diseñar un sistema modular para biblioteca.
