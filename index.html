1. Clases Abstractas e Interfaces

Las clases abstractas y las interfaces son conceptos fundamentales de la Programación Orientada a Objetos. Permiten aplicar abstracción y polimorfismo, facilitando la reutilización del código y la organización de sistemas complejos.

¿Qué es una clase abstracta?

Una clase abstracta es una clase que no puede ser instanciada directamente y sirve como base para otras clases. Puede contener métodos abstractos y métodos normales.

Ejemplo 1 - Clase Abstracta


abstract class Animal {

    abstract void sonido();
}

class Perro extends Animal {

    void sonido() {
        System.out.println("Guau");
    }
}

public class Main {

    public static void main(String[] args) {

        Perro perro = new Perro();
        perro.sonido();
    }
}

¿Qué es una interfaz?

Una interfaz define comportamientos que otras clases deben implementar. En Java se utiliza la palabra reservada interface.

Ejemplo 2 - Interfaz


interface Vehiculo {

    void acelerar();
}

class Carro implements Vehiculo {

    public void acelerar() {
        System.out.println("El carro acelera");
    }
}

public class Main {

    public static void main(String[] args) {

        Carro carro = new Carro();
        carro.acelerar();
    }
}
