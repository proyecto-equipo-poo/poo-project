1. Clases Abstractas e Interfaces

Las clases abstractas y las interfaces son pilares fundamentales de la Programación Orientada a Objetos porque permiten aplicar abstracción y polimorfismo.

Ejemplo 1


abstract class Animal {

    abstract void sonido();
}

class Perro extends Animal {

    void sonido() {
        System.out.println("Guau");
    }
}

Ejemplo 2


interface Vehiculo {

    void acelerar();
}

class Carro implements Vehiculo {

    public void acelerar() {
        System.out.println("Acelerando...");
    }
}
